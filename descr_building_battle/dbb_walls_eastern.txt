; ******** wall and tower animations
; ******** Kill zones go after duration line in transition scripts.  You can have 
; ******** as many as you like.  Defined as 'kill_zone <tick> <cas>',
; ******** eg 'kill_zone 20 my_kill_zone.cas', root directory is models_building/kill_zone.
; Params for dust_anim command:
; 	<start_tick> dust_anim <duration_ticks> <size> <quantity> <frequency> <start_anim_time> <end_anim_time>
; Starting from start_tick, until start_tick + duration_ticks, every 'frequency' ticks this will add
; 'quantity' dust clouds of radius 'size', at positions determined by the positions of geometries at the
; appropriate time in the animation.  It picks geometries at random, and the time in the animation is
; defined separately - ie it doesn't have to be the current positions, so you can create dust late or 
; early if it looks wrong.
; Wall damage:  collapse in five seconds, hold for two, then fade out over five.

; ******** wall and tower animations
; ******** Kill zones go after duration line in transition scripts.  You can have 
; ******** as many as you like.  Defined as 'kill_zone <tick> <cas>',
; ******** eg 'kill_zone 20 my_kill_zone.cas', root directory is models_building/kill_zone.
; Params for dust_anim command:
; 	<start_tick> dust_anim <duration_ticks> <size> <quantity> <frequency> <start_anim_time> <end_anim_time>
; Starting from start_tick, until start_tick + duration_ticks, every 'frequency' ticks this will add
; 'quantity' dust clouds of radius 'size', at positions determined by the positions of geometries at the
; appropriate time in the animation.  It picks geometries at random, and the time in the animation is
; defined separately - ie it doesn't have to be the current positions, so you can create dust late or 
; early if it looks wrong.
; Wall damage:  collapse in five seconds, hold for two, then fade out over five.

; ******** wall and tower animations
; ******** Kill zones go after duration line in transition scripts.  You can have 
; ******** as many as you like.  Defined as 'kill_zone <tick> <cas>',
; ******** eg 'kill_zone 20 my_kill_zone.cas', root directory is models_building/kill_zone.
; Params for dust_anim command:
; 	<start_tick> dust_anim <duration_ticks> <size> <quantity> <frequency> <start_anim_time> <end_anim_time>
; Starting from start_tick, until start_tick + duration_ticks, every 'frequency' ticks this will add
; 'quantity' dust clouds of radius 'size', at positions determined by the positions of geometries at the
; appropriate time in the animation.  It picks geometries at random, and the time in the animation is
; defined separately - ie it doesn't have to be the current positions, so you can create dust late or 
; early if it looks wrong.
; Wall damage:  collapse in five seconds, hold for two, then fade out over five.
